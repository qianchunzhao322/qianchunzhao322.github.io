CSS特效——旋转相册

一、制作思路

想通过定位的方式将所有的相片叠加到一起，再通过3D旋转将所有的相册拼成一个正方体，最后使用动画是其动起来。附加特效：：：可以鼠标经过是，相片的映像层有小变大，由虚变实。

二、步骤详解

2、设置相册拼成正方体

    left: 50px;
    top: 50px;
    height: 150px;
    width: 150px;

    transform: translateZ(80px);

}

}

3、使其旋转起来

@keyframes rotateY {
    0%{
        transform: rotateY(0deg);
        
    }
    100%{
        
        transform: rotateY(360deg);
    }
    
}
